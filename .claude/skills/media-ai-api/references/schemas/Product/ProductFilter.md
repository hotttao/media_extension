# ProductFilter

Query filters for listing products.

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `targetAudience` | enum: MENS, WOMENS, KIDS | No | Filter products by target audience. |
| `search` | string | No | Keyword used to search product names and metadata. |
| `tags` | string[] | No | Filter products by tags. |
